# EntityCore
Vulkan-based engine used to simplify use of Vulkan and debugging.

# Requirement
Mandatory :
  - Latest Vulkan SDK installed
  - Vulkan 1.1 support
  - SDL2
  - stb_image.h (http://nothings.org/stb)

Recommended :  
  - Latest vulkan driver (20 February 2021 for VK_KHR_synchronisation_2)
